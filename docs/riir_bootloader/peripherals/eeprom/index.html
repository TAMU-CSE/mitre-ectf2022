Module riir_bootloader::peripherals::eeprom

source · [−]
Expand description

Provides utilities and wrapper types for working with EEPROM memory. Refer to EepromLayout for an overview of how EEPROM memory is allocated.

Wrapper to encrypt the inner type in EEPROM.

Flag

Opaque wrapper for a boolean flag. Internally, the bit patterns for true and false have a maximally large Hamming distance to make error detection easier.

FlashKey

EEPROM sector containing data for decrypting firmware, configuration, and release message packages, which are all encrypted with crypto_secretstream. This is stored encrypted.

FwFlag
